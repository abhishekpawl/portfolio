
## ECMAScript
- scripting language specification on which JavaScript is based

## JavaScript
- scripting language that conforms to the ECMAScript specification
- *beyond ECMAScript*
	- includes additional features - Document Object Model (DOM) manipulation

## Common JS Browser engines
- V8 - used by google chrome/chromium
- SpiderMonkey - used by Firefox

## NodeJS
- runtime
- took out V8 engine, added some Backend functionalities (filesystem reads) on top to create this runtime

## Bun
- runtime
- NodeJS is slow
- significantly faster runtime
- written in Zig

# HTTP Server

- Hyper Text Transfer Protocol
- protocol defined for machines to communicate

## HTTP (Client Side)
- Protocol (HTTP/HTTPS)
- Address (URL/IP/Ports)
- Route
- Header, Body, Query Parameters
- Method

## HTTP (Server Side)
- Response Header
- Response Body
- Status Codes

## Common Methods
- GET
- POST
- PUT
- DELETE

## Common Status Codes
- 200 - Everything is OK
- 404 - Page/Route not found
- 403 - Authentication issues
- 500 - Internal server error
